# kazakhScript
kazakhScript - ЯП-транслятор в JavaScript 


Принцип работы: хз, я скоммуниздил код транслятора и конвертера из YoptaScript и заменил только лексемы.


Описание файлов:  

  converter.html и converter.js - конверторы из JS в KS
  
  kazakhlib.js - библиотека лексем
  
  kazakhscript.js - транслятор
